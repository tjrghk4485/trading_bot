# 🚀 KIS 실시간 급등주 자동매매 시스템

한국투자증권(KIS) OpenAPI를 활용하여 실시간으로 시장을 스캔하고, 거래량이 급증하는 종목을 자동으로 매수하여 목표 수익률(+3%) 달성 시 매도하는 파이썬 데몬 스크립트입니다.

## 📌 핵심 기능
- **실시간 종목 스캔**: `volume-rank` API를 통해 거래량/상승률 상위 종목을 실시간 모니터링.
- **필터링 조건**: 
  - 시가총액 1,000억 원 이상의 안정적인 종목.
  - 전일 대비 거래량 증가율 200% 이상 포착.
- **자동 매매**: 조건 충족 시 시장가로 즉시 매수.
- **익절 및 종료**: 매수 평단가 대비 **+3.0%** 수익 시 전량 매도 후 프로그램 자동 종료.
- **보안**: API 키와 계좌 정보는 `.env` 파일에서 관리하여 코드 유출 방지.
- **로깅**: 모든 매매 과정과 수익률 변화를 `trading_log.txt`에 상세히 기록.

## 🛠️ 설치 및 설정

### 1. 필수 라이브러리 설치
파이썬이 설치된 환경에서 아래 명령어를 실행하세요.
```bash
pip install requests python-dotenv
```

### 2. 환경 변수 설정 (.env)
프로젝트 루트 폴더에 생성된 `.env` 파일을 열고 본인의 API 정보를 입력하세요.
- `KIS_APP_KEY`: 한국투자증권 앱 키
- `KIS_APP_SECRET`: 한국투자증권 앱 시크릿
- `KIS_CANO`: 계좌번호 (앞 8자리)
- `KIS_URL`: 실전투자 또는 모의투자 주소

### 3. 프로그램 실행
```bash
python trading_bot.py
```

## 📈 매매 전략 상세
1. **탐색 (Scan)**: 5초 주기로 거래량 상위 30개 종목을 검사합니다.
2. **검증 (Filter)**: 시가총액 조건과 거래량 급증 조건을 모두 만족하면 매수 대상으로 선정합니다.
3. **매수 (Buy)**: 시장가(01) 주문으로 즉시 체결을 우선시합니다.
4. **감시 (Monitor)**: 2초 주기로 현재가를 조회하여 수익률을 계산합니다.
5. **매도 (Sell)**:
   - **익절**: 수익률 >= +3.0%
   - **손절**: 수익률 <= -2.0% (기본 설정)
   - 매도 성공 후 로그를 남기고 시스템을 종료합니다.

## ⚠️ 주의 사항
- **투자 책임**: 본 스크립트는 투자 참고용이며, 모든 투자의 책임은 사용자 본인에게 있습니다.
- **API 제한**: 한국투자증권 API는 초당 호출 제한이 있습니다. 과도한 수정 시 차단될 수 있습니다.
- **테스트**: 실전 투자 전 반드시 **모의투자** 환경에서 충분히 검증하세요.
